# Experiment 8 - Fullstack Authentication

## Features
- React Login Form
- JWT Authentication
- Protected Routes
- Role-Based Access Control (RBAC)

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## Setup
1. Install dependencies for client and server
2. Set up MongoDB and add .env file
3. Run server: `node server.js`
4. Run client: `npm start`
